### 🔹 Summary
- Oracle RAC + Active Data Guard 환경에서 MAA 구성을 위한 `tnsnames.ora` 설계 문의
  - `FAILOVER=ON`, `LOAD_BALANCE=ON` 상태에서 Standby 접속으로 인한 DML 실패 위험성 언급
  - TNS 파일을 2개로 나누는 경우 무중단 전환 어려움 → 이를 해결할 구성 예제 모색

- PDF 인쇄 시 OCR 정보 제거 방법 문의
  - PowerPoint만으로 워터마크 추가 가능한지 확인
  - 약 200페이지의 PDF를 처리해야 하는 상황

- Oracle `impdp` 시 `EXCLUDE=SCHEMA` 구문 사용 방법 문의
  - `EXCLUDE=SCHEMA:"='SYS','SYSTEM'"`과 같은 구문 예시 제공
