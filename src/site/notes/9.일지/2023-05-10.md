- 코드리뷰..
	- 얼개가 완성되지 않은 상태의 코드리뷰는 어렵구나..

```
윈도우 cmd 창에서 한글이 깨지는 것은 보통 유니코드(Unicode) 인코딩이 제대로 처리되지 않아 발생합니다. 이를 해결하기 위해서는 다음과 같은 방법들을 시도해 볼 수 있습니다.

1.  윈도우 cmd 창에서 한글 인코딩을 변경하기 cmd 창에서 `chcp 65001`을 입력하여 인코딩을 UTF-8로 변경합니다. 이는 한글을 포함한 모든 유니코드 문자를 지원합니다.
    
2.  글꼴 변경하기 cmd 창의 기본 글꼴이 한글을 지원하지 않을 수 있습니다. 이 경우, 글꼴을 바꿔서 한글을 지원하는 글꼴로 변경하면 됩니다. cmd 창을 마우스 오른쪽 버튼으로 클릭하고 "속성"을 선택합니다. "글꼴" 탭에서 한글을 지원하는 글꼴로 변경합니다.
    
3.  PowerShell 사용하기 PowerShell은 cmd 창보다 더 많은 기능과 유니코드 지원을 제공합니다. cmd 창에서 PowerShell을 실행하여 한글이 깨지지 않는 환경에서 작업할 수 있습니다.
    
4.  다른 텍스트 에디터 사용하기 cmd 창 대신 다른 텍스트 에디터(예: VSCode, Sublime Text 등)를 사용하여 작업할 수 있습니다. 이들 텍스트 에디터는 유니코드를 완벽하게 지원하며, cmd 창보다 더욱 편리한 작업 환경을 제공합니다.
```
